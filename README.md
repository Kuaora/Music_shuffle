# Music Shuffle

Программа для случайного воспроизведения музыки из папки.

## Описание

Music Shuffle — это простой Python-плеер, который автоматически находит все MP3 файлы в указанной папке, перемешивает их в случайном порядке и воспроизводит один за другим.

## Возможности

- Автоматический поиск всех MP3 файлов в папке
- Случайное воспроизведение треков
- Настраиваемая громкость
- Последовательное воспроизведение всех треков
- Вывод информации о текущем треке в консоль

## Требования

- Python 3.x
- pygame

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Kuaora/Music_shuffle.git
cd Music_shuffle
```

2. Установите необходимые зависимости:
```bash
pip install pygame
```

## Использование

1. Откройте файл `main.py` и укажите путь к вашей папке с музыкой в строке 9:
```python
os.chdir("C:/Путь/к/вашей/папке/с/музыкой")
```

2. Запустите скрипт:
```bash
python main.py
```

3. Программа начнет воспроизводить треки в случайном порядке. В консоли будет отображаться название текущего трека.

## Настройка

### Изменение громкости

Громкость можно настроить в строке 14 (значение от 0.0 до 1.0):
```python
pygame.mixer.music.set_volume(0.2)  # 0.2 = 20% громкости
```

### Поддерживаемые форматы

По умолчанию программа работает с MP3 файлами. Для добавления других форматов измените строку 11:
```python
for song in glob('*.mp3'):  # можно заменить на *.wav, *.ogg и т.д.
```

## Как это работает

1. Программа сканирует указанную папку на наличие MP3 файлов
2. Создает список всех найденных треков
3. Перемешивает список в случайном порядке
4. Последовательно воспроизводит каждый трек
5. Ожидает завершения текущего трека перед запуском следующего

## Примечания

- Убедитесь, что путь к папке с музыкой указан правильно
- Программа воспроизводит все треки из папки последовательно
- Для остановки воспроизведения закройте программу (Ctrl+C)
